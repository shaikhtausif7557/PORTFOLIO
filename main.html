Document Management System

Project Overview: The Document Management System is a web application developed using React.js and Firebase. The goal of this project is to provide users with a convenient platform for uploading, managing, and downloading files. With this system, users can easily store their documents in a secure cloud-based storage solution and access them whenever needed.

Technologies Used:

A JavaScript library for building user interfaces.

Firebase: A cloud-based platform that offers various services, including real-time database and storage.

Key Features:

Upload: Users can upload their documents to the system by selecting the files from their local storage. The system securely stores the files in the Firebase cloud storage.

File Management: The system provides a user-friendly interface for managing uploaded files. Users can view the list of uploaded documents, organize them into folders or categories, and perform operations like renaming or deleting files.

File Download: Users can easily download their uploaded files with just a few clicks. The system retrieves the requested file from the Firebase storage and initiates the download process.

User Authentication:The system incorporates user authentication using Firebase Authentication. This ensures that only authorized users can access and manage their uploaded documents.

Charity Website with Payment Gateway (React.js and MongoDB)

Project Overview:

The Charity Website with Payment Gateway is a full-stack web application developed using React.js and MongoDB. The primary goal of this project is to provide a platform for users to donate to various charitable causes online. The website incorporates a payment gateway for secure transactions and utilizes MongoDB for data storage and retrieval.

Technologies Used:

React.js: A JavaScript library for building user interfaces.

MongoDB: A NoSQL database for storing and managing data.

Express.js: A web application framework for Node.js used for server-side development.

Node.js: A JavaScript runtime for server-side development.

Payment Gateway: Integrated a secure payment gateway to facilitate online donations.

Key Features:

Donation Form: The website includes a donation form where users can enter their details, including name, email address, donation amount, and the cause they wish to support. The form data is securely stored in the MongoDB database.

Payment Gateway Integration: A secure payment gateway is integrated into the website to handle online transactions. Users can make their donations using various payment methods, such as credit/debit cards or digital wallets, with the assurance of secure and encrypted transactions.

Cause Information: The website provides comprehensive information about each charitable cause, including details about the organization, its mission, and the impact of donations. Users can make informed decisions about the causes they want to support.

User Authentication: Users can create accounts and log in to the website. Authenticated users have the ability to view their donation history, manage their personal information, and easily track their contributions.
